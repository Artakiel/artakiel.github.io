## 🚀 Publier sur GitHub Pages

1. Créer un dépôt GitHub nommé `osnit-live`
2. Uploader le fichier `index.html`
3. Aller dans **Settings → Pages**
4. Source : **Deploy from a branch → main → / (root)**
5. Votre lien est actif dans 1 à 2 minutes

## 🔧 Activer le badge LIVE manuellement

Quand vous commencez un live, ouvrez la console du navigateur sur votre page et tapez :

```javascript
// Pour activer (remplacer 'twitch' par 'kick' ou 'youtube')
setLiveStatus(true, 'twitch')

// Pour désactiver
setLiveStatus(false)
```
